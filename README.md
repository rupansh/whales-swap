# Whales-Swap

## Compile

`pnpm run build`

Markdown docs are automatically generated in the `./docs` folder  
Typings are auto generated in `./typechain-types`

## Deploy

setup `.env`

`pnpx hardhat run scripts/<script_name>.ts --network <network>`

see hardhat.config.ts for networks

## Tests

`pnpm run test`
